01

Website Redesign

Re-Design
Webseite (FR)

Laufend · 2024–2025

Die bestehende Webseite war vier Jahre alt und benötigte einen grundlegenden optischen Refresh. Als Pilot wurde die .fr-Domain gewählt — mit wenigen Seiten und geringem Traffic bleibt potenzielle Fehlertoleranz auf Kundenseite minimal.

Tailwind CSS als Grundstruktur implementieren, gemeinsam mit klaren HTML-Anweisungen für das Content Team.
Markdown-Integration damit das Content-Team unabhängig von HTML-Strukturen arbeiten kann.
WordPress-Migration — weg von WP hin zu einer schlanken, performanten Lösung.
Kleineres CSS-Footprint & messbar bessere Performance-Scores.
Aufbau eines internen BOX-Systems als modulares Webseitenbaukasten-Framework.

02

Software · UI Redesign

Toolbox
Re-Design

Laufend · 2024–2025

Die Toolbox ist das Herzstück und Hauptprodukt des Unternehmens — eine komplexe Softwareplattform mit tiefen Datenbankstrukturen, an der ein cross-funktionales Team aus Backend, Frontend und Design gemeinsam arbeitet. Ziel war es, die Software nicht nur optisch zu modernisieren, sondern sie für Nutzer spürbar einfacher bedienbar zu machen.

Tailwind CSS implementieren und das bestehende CSS massiv verkleinern — weniger Overhead, mehr Konsistenz.
Visuelles Redesign — modernere Ästhetik, klarere Hierarchien und ein kohärentes Design System für das gesamte Produkt.
UX-Optimierung — komplexe Datenbankansichten vereinfachen und die Lernkurve für neue Nutzer reduzieren.
Besonderer Fokus auf Responsive Darstellung — nahtlose Nutzung auf allen Gerätegrößen als Kernanforderung.
Cross-Team Kollaboration — enge Abstimmung zwischen Backend, Frontend und Design in einem größeren Entwicklungsteam.
